Introduction

App Stats permet d'importer des fichiers Excel (.xlsx), de configurer vos colonnes, de filtrer les données et de produire des statistiques mensuelles, graphiques et rapports PDF.

Cette page s'ouvre dans un nouvel onglet pour que vous puissiez consulter l'aide tout en travaillant dans l'application.

Documentation mise à jour le 25/06/2025 — couvre sidebar piles/calculs, surfiltre, multi-graphiques, presets JSON et export PDF complet.

Pour qui ? Les sections « Guide du débutant » et les exemples conviennent aux nouveaux utilisateurs. Les sections détaillées sur les calculs, presets et graphiques s'adressent aux utilisateurs avancés.

Guide du débutant

Suivez ces étapes pour obtenir votre première statistique mensuelle en quelques minutes.

  1. Importer un fichier — Cliquez sur Importer un fichier .xlsx en haut à droite. Choisissez la ligne d'en-tête, sélectionnez les colonnes et validez.
  2. Configurer les dates — Si une colonne contient des dates, ouvrez Configurer les colonnes et définissez le formateur « Date » avec le bon format source.
  3. Filtre mensuel — Onglet Filtre mensuel : choisissez la colonne date de référence, créez une pile (+ Nouvelle pile) avec un filtre (ex. texte « contient »).
  4. Voir les résultats — Onglet Valeurs : le tableau affiche le nombre de lignes par pile et par mois (année courante par défaut). Cliquez une cellule pour voir le détail.
  5. Graphique — Onglet Graphiques : sélectionnez les piles (case V/G cochée), réglez la plage, puis Actualiser le graphique.
  6. Exporter — Exporter rapport PDF inclut le tableau Valeurs et tous vos sous-onglets graphiques.
Exemple concret Vous avez un fichier de tickets support avec une colonne « Statut » et une colonne « Date de clôture ».
1. Formatez « Date de clôture » en date.
2. Créez une pile « Tickets fermés » avec filtre texte : Statut contient « Fermé ».
3. Dans Valeurs, vous verrez combien de tickets fermés existent par mois.

Presets d'import

Dans la fenêtre d'import, la section Presets d'import permet de mémoriser la ligne d'en-tête, les colonnes cochées, les formateurs et les filtres du tableau actif. Enregistrez, rechargez ou exportez en JSON.

Les mêmes presets sont accessibles depuis l'en-tête (voir Preset import dans l'en-tête).

Colonnes et formatage

Via Configurer les colonnes, vous pouvez renommer les colonnes et appliquer un formateur :

  • Date — conversion avec format source (ex. DD/MM/YYYY HH:mm) et affichage personnalisé.
  • Nombre — normalisation pour filtres numériques et calculs.
  • Majuscules — uniformisation du texte.
Un mauvais format de date empêche le regroupement mensuel : vérifiez l'onglet Information si les dates semblent incorrectes.

Données

Affiche le tableau brut importé, avec défilement et recherche dans les lignes.

Information

Résumé du tableau : nombre de lignes/colonnes, types détectés, plages de dates et statistiques numériques.

Filtre courant

Les piles de filtres combinent plusieurs conditions pour isoler des lignes en temps réel.

Types de filtres

  • Texte (contient, commence par, finit par)
  • Valeur exacte (liste multi-sélection)
  • Date (avant, après, période)
  • Nombre (comparaisons et plages)
  • Vide / Non vide
  • Surfiltre — filtrer à partir des résultats d'une autre pile (voir ci-dessous)

Connecteurs

ET, OU, AVEC, SANS lient chaque filtre au précédent.

Actions utiles

  • Sidebar — liste des piles à gauche ; la pile sélectionnée s'édite à droite.
  • V/G — case à cocher pour afficher ou masquer la pile dans Valeurs et Graphiques.
  • ↑ ↓ — réordonner les piles.
  • Importer preset filtre (JSON) — charge un fichier preset exporté.
  • Presets filtres — historique local avec versions.

Sidebar des piles

Cliquez une pile dans la liste pour l'éditer. Le panneau principal (à droite) affiche uniquement l'élément sélectionné ; seule cette zone défile si le contenu est long — la sidebar reste fixe.

  • Bouton ⧉ dans l'éditeur : dupliquer la pile active.
  • Les résultats filtrés s'affichent sous les filtres (filtre courant uniquement).

Surfiltre

Type de filtre Surfiltre (pile source) : restreint les résultats aux lignes d'une autre pile, puis enchaînez d'autres filtres pour affiner par étapes.

Exemple surfiltre Pile A « Tous les clos » → Pile B avec surfiltre sur A + filtre texte « urgent » = tickets clos ET urgents.

Filtre mensuel

Comme le filtre courant, mais les comptages sont répartis par mois selon la colonne date choisie en haut de la vue.

Les filtres date peuvent être relatifs au mois affiché (début, fin, pendant le mois).

Sidebar piles et calculs

La sidebar liste toutes les piles et, en dessous, les calculs, compteurs et analyses dates. Cliquez un élément pour l'éditer dans le panneau principal. Les boutons de création sont dans l'en-tête de la vue :

  • + Nouvelle pile — pile de filtres mensuels
  • + Calcul — formule entre piles ou calculs
  • + Compteur — agrégation sur une pile filtrée
  • + Analyse dates — durées entre deux colonnes date

Case V/G sur les piles : contrôle la visibilité dans Valeurs et Graphiques (les calculs y apparaissent toujours). Utilisez ↑ ↓ pour réordonner piles et calculs librement dans la liste.

Calculs avancés

Calcul formule (+ Calcul)

Combine des piles ou d'autres calculs : addition, soustraction, multiplication, division, ratio, etc.

Compteur (+ Compteur)

Sur le résultat d'une pile filtrée, pour chaque mois :

  • Somme d'une colonne numérique
  • Somme du nombre de caractères d'une colonne texte
  • Nombre d'occurrences d'un caractère choisi
Exemple compteur Pile « Commandes validées » + compteur en mode « Somme numérique » sur la colonne « Montant » → total des montants par mois pour les commandes filtrées.

Analyse de dates (+ Analyse dates)

Entre deux colonnes date, sur les lignes d'une pile :

  • Moyenne, minimum ou maximum de durée en jours
  • Nombre de lignes dont la durée est dans une plage (ex. entre 3 et 10 jours)

Valeurs et export

Matrice croisée : piles / calculs en lignes, mois en colonnes. Cliquez une cellule pour le détail des lignes.

  • Exporter XLSX — télécharge la matrice visible.
  • Dans le détail d'une cellule : Exporter XLSX pour les lignes filtrées.
  • Colonne PDF — inclure ou exclure une ligne du rapport PDF.

Plage d'affichage : totalité, par année ou par mois. Par défaut, l'année courante est sélectionnée si elle existe dans les données.

Graphiques

Visualisez les séries visibles dans l'onglet Valeurs (piles avec V/G coché + calculs).

  • Types : colonnes, lignes, camembert (min. 2 séries).
  • Plage : toutes les dates, année, mois ou plage personnalisée (date début / fin).
  • Couleurs et cases à cocher pour inclure ou exclure chaque série.
  • Cliquez Actualiser le graphique après chaque changement de paramètres.

Dispositions en colonnes

DispositionComportement
ParallèlesBarres côte à côte ; par série, choix Barres ou Ligne (courbe continue sur tous les mois).
Empilées (addition)Les valeurs s'additionnent verticalement par mois.
Empilées (soustraction)La 1re série s'additionne, les suivantes se soustraient visuellement.

Paramètres d'affichage

  • Max ordonnée — fixe l'échelle verticale (vide = automatique).
  • Position légende — droite, gauche, haut ou bas.

Édition des valeurs

  • Liste Valeurs du graphique — valeurs actives ; bouton × pour retirer.
  • Menu déroulant Ajouter une valeur — choix parmi les piles/calculs disponibles (V/G).
  • Panneau Couleurs et styles par valeur — couleur et type barres/ligne par série.

Plusieurs graphiques

Bouton + Ajouter un graphique : sous-onglets avec noms éditables (double-clic). Chaque graphique a sa propre plage, type, disposition et séries.

Export PDF Tous les sous-onglets graphiques sont inclus dans le rapport PDF, chacun sous son titre. Pensez à actualiser les graphiques avant l'export pour des captures à jour.

Presets et sauvegarde

Presets filtres

Bouton Presets filtres (Filtre courant ou mensuel) :

  • Nommer et enregistrer la configuration actuelle (piles, calculs, colonne date mensuelle)
  • Historique local (30 entrées max) avec date et numéro de version
  • Charger une version antérieure ou mettre à jour un preset existant
  • Exporter fichier JSON depuis la modal — télécharge la config complète

Bouton Importer preset filtre (JSON) (même vue) : charge directement un fichier JSON exporté, sans passer par le presse-papiers. Le preset inclut filtres, calculs, ordre sidebar, plage Valeurs et configuration des graphiques (sous-onglets, couleurs, séries).

L'export/import JSON via presse-papiers n'est plus disponible dans les vues filtres : utilisez les fichiers JSON.

Presets d'import

Dans la fenêtre d'import ou via l'en-tête (voir ci-dessous).

Preset import dans l'en-tête

À côté de « Importer un fichier .xlsx » :

  • Exporter preset import — télécharge un fichier JSON (colonnes, formateurs, filtres).
  • Importer preset import — charge un JSON preset ; appliqué à la modal d'import ou aux filtres du tableau actif.

Rapport PDF

Exporter rapport PDF (en-tête) génère une page d'impression contenant :

  1. Le tableau des valeurs selon la plage affichée dans l'onglet Valeurs (lignes cochées PDF uniquement).
  2. Tous les graphiques configurés — un bloc par sous-onglet, avec le titre du graphique.

Les graphiques sont rendus automatiquement à l'export (pas besoin d'être sur l'onglet Graphiques). Les séries et plages de chaque sous-onglet sont respectées.

FAQ

Pourquoi l'onglet Valeurs est vide ?

Vérifiez qu'une colonne date est sélectionnée dans Filtre mensuel et qu'au moins une pile existe. Les dates doivent être correctement formatées à l'import.

Le graphique ne se met pas à jour tout seul

C'est normal : cliquez « Actualiser le graphique » après avoir modifié les paramètres (performance).

Comment partager ma configuration de filtres ?

Dans Filtre courant ou mensuel : exportez un fichier JSON via Presets filtres → « Exporter fichier JSON », ou demandez au destinataire d'utiliser Importer preset filtre (JSON) pour charger votre fichier.

Une pile n'apparaît pas dans Valeurs ou Graphiques

Vérifiez la case V/G dans la sidebar du filtre mensuel. Si elle est décochée, la pile est masquée de la matrice et des graphiques.

Le PDF ne contient qu'un seul graphique

Seuls les sous-onglets créés via « + Ajouter un graphique » sont exportés. Un seul onglet = un seul graphique dans le PDF.

Que fait le bouton Dupliquer (⧉) sur une pile ?

Crée une copie identique de la pile avec « copie » ajouté au nom, utile pour tester des variantes.

Différence entre filtre courant et filtre mensuel ?

Le filtre courant explore les données en direct. Le filtre mensuel alimente les comptages par mois, les calculs et les graphiques.

Où sont stockés les presets ?

Dans le navigateur (localStorage). Pensez à exporter en JSON pour sauvegarder sur un autre poste.
