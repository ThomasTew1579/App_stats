creer une app web complete en js vanilla, html et css
descitption de l'app:
- le but est d'importer des fichier .xlsx puis de les mapper dans une interface web avant de les analyser avec la librairie que j'ai mis dans le répertoire
- l'interface est repreésenté par un tableau de donnée
- posibilité d'avoir plusieur tableau accessible depuis une bar d'onglet
- possibilité de modifier le nom des tableau 
- possibilité d'ajouter les info d'un fichier a un tableau existant en utilisant le meme mapping
- le mapping permet de choisir les information du fichier a afficher : 
    - a l'import d'un fichier, une modal contextuel s'ouvre et contient :
        - un selecteur qui permet de choisir si le fichier doit etre importer dans un nouveau tableau ou a ajouter dans un tableau deja existant ou de merge un tableau avec un rapport du nombre de ligne importé, le nombre de ligne dont des veleur ont été remplacé en se basant sur la colonne ID
        - un selecteur de ligne pour définir la ligne de titre des colonnes, si la ligne selectionné est la ligne 2, la ligne 1 ne sera pas prise en compte
        - une liste des information contenu dans les colonnes avec un systeme de case a cocher pour definir les colonnes a afficher dans l'interface 
        - pour chaque colonnes selectionnées :
            - un selecteur qui permet de lier chaque colonne a celle existante dans le cas d'un ajout a un tableau exitant 
            - si c'est un nouveau tableau, un systeme de formateur d'information : 
                - un formateur de string en date qui fonctionnerait de cette façon : si l'infromation dans la colonne est sous cette forme "03/10/2025 13:24", permettre de choisir le mapping des information temporel de la string sous cette frome "DD/MM/YYYY HH:mm" qui sera convertie en date en TS puis affiché de differentes façon au choix
                - un formateur de string en nombre et retourner un message d'avertissement si une des valeur ne sont pas convertie
                - un formateur pour transformer des string en uppurcase
        - un bouton pour importer 
- la posibiliter de modifier le mapping et le formatage de chaque colonnes meme apres l'import
- pour chaque tableau, une page de statistique qu'on fera dans un second temps
- ajoute du style simple et sombre pour toute l'interface

CONNECTEURS IMPLEMENTES
-------------------------
| Connecteur | Comportement |
|------------|--------------|
| ET         | Intersection : la ligne doit correspondre au filtre ET aux précédentes |
| OU         | Union : la ligne peut correspondre à ce filtre ou aux résultats précédents |
| AVEC       | La ligne doit avoir une valeur non vide dans la colonne ET correspondre au filtre |
| SANS       | Exclusion : les lignes correspondant à ce filtre sont retirées du résultat |

DETAILS TECHNIQUES
------------------
- Propriété `connector` sur chaque filtre (défaut : ET)
- Sélecteur affiché entre deux filtres dans la pile
- Badge du connecteur visible dans l'en-tête de chaque filtre (sauf le 1er)
- Export/import JSON : le connecteur est inclus dans chaque filtre
- Compatibilité : filtres existants sans connecteur → ET par défaut
